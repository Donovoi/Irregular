|Name                                                   |Description                           |IsGenerator|
|-------------------------------------------------------|--------------------------------------|-----------|
|[?<RegularExpression_Group>](Group.regex.txt)          |Matches groups in a regular expression|False      |
|[?<RegularExpression_Quantifier>](Quantifier.regex.txt)|Matches a quantifier                  |False      |
